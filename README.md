# Global Investment Simulator

A single-page React app (Vite + TypeScript + Tailwind) that builds an auto-allocated,
globally diversified portfolio (USA + India) with charts, stress tests, and CSV export.

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy
- Vercel / Netlify / Cloudflare Pages
- Build command: `npm run build`
- Output directory: `dist`
